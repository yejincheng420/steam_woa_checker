# 🎮 Steam ARM on Windows 兼容性检测工具

一个轻量、极速、免登录的本地开源工具。自动读取你的本地 Steam 游戏库，并与 [WorksOnWoa](https://www.worksonwoa.com/) 官方数据库进行比对，一键获知你的游戏在 Windows on ARM 设备（如骁龙 X Elite）上的运行状态。好吧，也支持epic。

## ✨ 核心特性

* 🚀 **极速本地扫描**：无需输入 Steam 账号密码，安全读取本地 `.acf` 文件获取已安装游戏列表。
* 📦 **离线缓存架构**：首次运行瞬间拉取 GitHub 云端全量数据并本地持久化，后续使用断网可用，告别卡顿。
* 🔍 **智能模糊匹配**：内置字符串正则化与容错算法，完美抹平商标符号（™/®）和拼写差异。
* 💻 **免环境独立运行**：提供打包好的单文件 `.exe`，小白玩家下载即用，无需配置 Python 环境。

## 📥 如何使用

1. 在右侧的 **Releases** 页面下载最新的 `SteamWoaChecker.exe`。
2. 双击运行软件。
3. （首次使用）点击 **“更新云端数据库”** 获取最新兼容性列表。
4. 点击 **“添加 Steam 库”**，选择包含 `steamapps` 的文件夹（通常是你的 Steam 安装目录或额外的游戏盘根目录）。
5. 点击 **“开始检测本地游戏”**，即可查看兼容性结果！

## 📊 状态说明

* 🟢 **Perfect / Playable / Runs**: 可以在 ARM 设备上流畅或降低画质运行。
* 🔴 **Unplayable**: 存在反作弊不兼容或指令集转译失败等问题。
* ⚪ **Unknown**: 游戏较冷门，官方数据库暂未收录测试结果。
